Приложение предназначено для получения частот слов / биграмм во фрагментах текстов, а также term-text матрицы для выбранных слов и их словоформ. Фрагменты текстов, по которым можно получить статистику - предложения, содержащие заданные на вкладке "фрагменты" слова. Поиск словоформ работает только для русского языка, хотя получить частоты слов / биграмм / term-text матрицу можно и для других языков.

1. На вкладке "Тексты" выберите текстовые файлы для загрузки. Кодировка файла для русского языка должна быть UTF-8.

2. На вкладке "Фрагменты" укажите слова, по которым будет произведена выборка предложений из загруженных текстов. На вкладке можно получить словоформы слов для поиска в текстах. Также можно экспортировать найденные предложения в txt файл.

3. На вкладке "Статистика" можно получить и экспортировать в txt файл частоты слов и биграмм. В текстовое поле "основы биграмм" можно ввести все возможные слова, которые будут первым словом найденных биграмм. Для основы биграмм можно получить словоформы с помощью кнопки "Морфология для биграмм". На вкладке "статистика" также можно экспортировать term-text матрицу с найденными словами / выборкой предложений. Термы для матрицы можно выбрать с помощью чекбокса "использовать". Также можно использовать словоформы терма - чексбокс "словоформы". Выбрать, какие именно словоформы вы хотите использовать для матрицы, можно с помощью кнопки формы - откроется модальное окно со всеми словоформами слова.
